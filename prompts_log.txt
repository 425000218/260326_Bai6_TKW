RESPONSIVE CSS CHEAT SHEET (2026 Edition)
1. Thẻ Meta bắt buộc (Trong file HTML)
AI đôi khi quên dòng này, khiến mọi code CSS Responsive đều vô nghĩa trên điện thoại:
HTML
<meta name="viewport" content="width=device-width, initial-scale=1.0">

2. Các điểm ngắt (Breakpoints) tiêu chuẩn
Thay vì viết số lộn xộn, hãy dạy sinh viên dùng các mốc chuẩn này:
CSS
/* 1. Mobile First (Mặc định cho điện thoại) */
body { font-size: 16px; }

/* 2. Tablet (Màn hình từ 768px trở lên) */
@media (min-width: 768px) {
    .container { width: 90%; }
    .sidebar { display: block; }
}

/* 3. Laptop/Desktop (Màn hình từ 1024px trở lên) */
@media (min-width: 1024px) {
    .container { max-width: 1200px; }
    .nav-menu { display: flex; }
}

3. Công thức "Vàng" cho Layout linh hoạt
A. Hình ảnh không bao giờ bị tràn (Responsive Images)
Dặn sinh viên luôn kiểm tra xem AI có dòng này không:
CSS
img {
    max-width: 100%;
    height: auto;
    display: block;
}

B. Grid tự động chia cột (Không cần Media Query)
Đây là kỹ thuật "thượng thừa" mà sinh viên nên yêu cầu AI sử dụng:
CSS
.grid-container {
    display: grid;
    /* Tự động chia cột: tối thiểu 250px, tối đa chiếm hết chỗ trống */
    grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
    gap: 20px;
}

C. Flexbox căn giữa đa thiết bị
CSS
.flex-center {
    display: flex;
    justify-content: center; /* Căn giữa ngang */
    align-items: center;     /* Căn giữa dọc */
    flex-wrap: wrap;        /* Tự xuống dòng trên màn hình nhỏ */
}
